# SEGA-Master-System-Controller
This is a recreation of the SEGA Master System Controller 
I've included Gerbers for PCB's, STL's, as well as CAD Models for Both.
I'm new to Github so still figuring things out lol. I'm currently working on a motherboard replacement similar to the opentendo project except for the SEGA Master System. 
On the PCB, I designed as a single but you can easily snap or cut it in two. Due to the design of this controller, it needs to be two seperate PCB's because they are at different heights. Also I rec that you get ENIG plating for the button contacts.
